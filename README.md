# الذراع البرمجي 🤖

تطبيق Android مبني بالكامل بالذكاء الاصطناعي — Kotlin + Jetpack Compose + Material3

## المميزات
- 🔐 مولّد هاش SHA-256 و MD5
- 📦 تشفير وفك تشفير Base64
- 🔑 مولّد كلمات مرور قوية
- 🔤 محوّل ASCII/Unicode

## البناء
GitHub Actions يبني ويوقّع الـ APK تلقائياً عند كل push.

## التحميل
اذهب إلى [Actions](../../actions) → آخر workflow → Artifacts → حمّل الـ APK الموقّع.
